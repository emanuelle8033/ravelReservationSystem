# TravelReservationSystem

# 🛫 Sistema de Reservas de Viagem

Sistema para gerenciamento de reservas de viagens, passageiros e destinos.

## 📋 Funcionalidades

- **Cadastro de Passageiros** - Gerenciar informações de passageiros
- **Gerenciamento de Destinos** - Cadastrar e listar destinos de viagem
- **Reservas de Viagens** - Criar e controlar reservas de viagens
- **Status da Viagem** - Acompanhar status (Agendada, Confirmada, Cancelada, Concluída)

## 🛠️ Tecnologias

- **Java** 26
- **Spring Boot** 3.x
- **Spring Data JPA**
- **MySQL**
- **REST API**
- **Swagger** (Documentação)

## 📦 Como Executar

1. Clone o repositório
2. Configure o banco de dados MySQL
3. Execute o projeto
4. Acesse:
   - **API Passageiros**: http://localhost:8080/api/passageiros
   - **API Destinos**: http://localhost:8080/api/destinos
